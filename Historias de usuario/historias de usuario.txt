Épica 1: Acceso y gestión de cuenta de usuario

HU 1:  Registrarse en la aplicación

> Como usuario quiero poder registrarme en Instagram Lite

Criterios de aceptación:
- Debe existir una opción para registrarse.
- El sistema debe permitir ingresar email/usuario y contraseña.
- Los campos obligatorios deben validarse.
- Con datos válidos, el registro debe completarse correctamente.
- Ante datos inválidos, se debe mostrar un mensaje de error.

HU 2: Iniciar sesión

> Como usuario quiero poder iniciar sesión en mi cuenta

Criterios de aceptación
- Deben existir campos para usuario/email y contraseña.
- El sistema debe permitir ingresar credenciales válidas.
- Al iniciar sesión correctamente, se debe mostrar el feed principal.
- Ante credenciales incorrectas, debe mostrarse un mensaje de error.
- El acceso debe restringirse si los campos están vacíos.
