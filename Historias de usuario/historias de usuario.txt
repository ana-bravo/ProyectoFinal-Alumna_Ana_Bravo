Épica 1: Acceso y gestión de cuenta de usuario

HU 1:  Registrarse en la aplicación

> Como usuario quiero poder registrarme en Instagram Lite

Criterios de aceptación:
- Debe existir una opción para registrarse.
- El sistema debe permitir ingresar email/usuario y contraseña.
- Los campos obligatorios deben validarse.
- Con datos válidos, el registro debe completarse correctamente.
- Ante datos inválidos, se debe mostrar un mensaje de error.

HU 2: Iniciar sesión

> Como usuario quiero poder iniciar sesión en mi cuenta

Criterios de aceptación
- Deben existir campos para usuario/email y contraseña.
- El sistema debe permitir ingresar credenciales válidas.
- Al iniciar sesión correctamente, se debe mostrar el feed principal.
- Ante credenciales incorrectas, debe mostrarse un mensaje de error.
- El acceso debe restringirse si los campos están vacíos.

HU 3: Recuperar contraseña

> Como usuario quiero poder recuperar mi contraseña, para volver a acceder a mi cuenta si la olvido

Criterios de aceptación
- Debe existir una opción visible de “¿Olvidaste tu contraseña?” en la pantalla de login.
- El sistema debe permitir ingresar un correo electrónico válido.
- Al enviar el correo, el sistema debe mostrar un mensaje de confirmación.
- El proceso debe permitir restablecer la contraseña y volver a iniciar sesión.

HU 4: Cerrar sesión (Logout)

> Como usuario quiero poder cerrar sesión de mi cuenta 

Criterios de aceptación
- Debe existir una opción visible para cerrar sesión.
- Al cerrar sesión, el usuario debe ser redirigido a la pantalla de login.
- No se debe poder acceder al contenido sin iniciar sesión nuevamente.


Épica 2: Visualización y publicación de contenido

HU 5: Visualizar el feed de publicaciones

> Como usuario quiero poder visualizar el feed de publicaciones

Criterios de aceptación
- Al ingresar a la aplicación, debe mostrarse el feed.
- Las publicaciones deben cargarse correctamente.
- Las imágenes deben visualizarse sin errores.
- Se debe poder hacer scroll para ver más publicaciones.

HU 6: Crear una publicación (post)

> Como usuario quiero poder publicar una imagen en mi perfil

Criterios de aceptación
- Debe existir un botón u opción para crear una publicación.
- El sistema debe permitir seleccionar una imagen del dispositivo.
- La publicación debe permitir agregar una descripción (opcional).
- Al confirmar, la publicación debe crearse correctamente.
- La publicación creada debe visualizarse en el perfil y/o feed.
- Ante un error, debe mostrarse un mensaje informativo.
